# Krepsys Frontend

React frontend for Krepsys newsletter reader.

## Tech Stack

- React 18
- Vite
- TailwindCSS
- React Router
- Axios
- React Query

## Development

```bash
npm install
npm run dev
```

Access at: http://localhost:18300

## Environment Variables

- `VITE_API_URL` - Backend API URL (default: http://localhost:8080)
